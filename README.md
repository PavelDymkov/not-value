# not

The **not** function is an alphabetic analogue of the unary operator **!**.
It is used to increase the readability of the code.

For example:

```js
import not from "not-value";

if(not("foo".includes("bar"))) {
    console.log("baz");
}
```

## Installation

```sh
$ npm install not-value
```
